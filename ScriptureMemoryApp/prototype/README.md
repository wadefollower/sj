# 背经 PWA

这是可直接部署的 PWA 版本。

## 本地预览

在 `prototype` 目录启动一个静态服务器：

```sh
python3 -m http.server 4173
```

然后打开：

```text
http://localhost:4173
```

不要用 `file://` 测试 PWA 安装和离线能力，因为 service worker 需要 `localhost` 或 HTTPS。

## iPhone 安装

1. 把 `prototype` 部署到 HTTPS 网站，例如 GitHub Pages。
2. 用 iPhone 的 Safari 打开网站。
3. 点分享按钮。
4. 选择“添加到主屏幕”。
5. 从主屏幕打开“背经”。

## 已支持

- 离线缓存应用外壳。
- 内置“五个确据 / A / B / C / D / E”经文。
- 经文库支持按“五个确据 / A / B / C / D / E / 自定义”筛选。
- 自定义经文保存在本机。
- 收藏、进度、语言、字号保存在本机。
- 中文和英文界面。
- 小到超大字号。
- 经文详情页支持浏览器朗读、停止朗读和语速选择。
- “听经文”页支持手动选择多节经文，并按顺序循环朗读。
- 自测模式需要点击确认后才显示原文，并提供逐字正确率。
- 内置经文包含中文正文和用户提供的 NIV 英文对照。
